    $tipo

    $tipo 

    Default

    property Tipo da linha lida do arquivo, conforme valores definidos no layout do arquivo de retorno. Tal valor é uma constante inteira que define se a linha é um header, trailer, linha detalhe, etc. Os valores específicos de tal atributo são definidos nas classes que herdam de @see RetornoBase, em métodos como getIdDetalhe().
